# Lifestyle Visualizer

Single-page interactive frontend that simulates playful body visualizations based on weight and lifestyle scenarios. Ready for deployment on GitHub Pages (place `index.html` at repository root).

Features:
- Inputs for height, weight, activity level, sport frequency, discipline, intermittent fasting and sugar restriction
- Scenario buttons for Week 0 / 4 / 8 / 12
- SVG-based avatar visualization (no photos)
- Simple client-side scenario engine (heuristic, not medical)
- Reset, randomize and export PNG

How to deploy
1. Commit these files to the repository root.
2. In GitHub repository settings -> Pages, serve from `main` branch root (or your repository default branch).

Notes
- This project intentionally does NOT provide medical advice. It shows imagined, illustrative archetypes only.
